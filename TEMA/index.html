Installation

  • Before you get started, make sure you have all the requirements available: Server space at a Hosting Provider, Domain Name, FTP client, Text Editor.

    • System: Unix, Linux or Windows. Unix is highly recommended.
    • Web server: Apache Web server 2.0 or later.
    • PHP 5.4 or later.
    • MySQL 5.0 or later.
    • At least 128 Mb of RAM on your server.

  • To install this theme you must have a working version of Prestashop already installed. For information in regard to installing the Prestashop platform, please see the Prestashop Wiki : http://doc.prestashop.com/display/PS17/Getting+started

  • After Prestashop is installed, you must delete “install” folder in your Prestashop.

Install

For each package, you will have: quickstart package, theme package, PSD. There are two ways to install a Prestashop Theme:

  • Install Quickstart Package: By using the quickstart package, you will set the theme exactly as our Demo with sample data.

  • Install Theme : If you have available shop, you can install only theme.


Enabling Static Block and Static Footer with other language

In your theme, if there are 3 languages or more, please enabling static block by below guide:

  • Login your prestashop admin
  • Navigate to International > Localization .
  • Choose tab Languages.
  • Navigate to Modules > Module Manager
  • Choose “Modules” tab
  • On Pos Staticblock or Pos Static Footer module, click "Configure" button.
  • Click "Edit" button of your selected block
  • Choose Source Code to show code HTML of this block in English. Then copy source code.
  • Click to choose language you want to translate. Choose Source Code
  • Click Save button when you finish

Static Blocks and Static Footer

Static Block

  1. How to add Static Block
    • Login your prestashop admin
    • Navigate to Modules > Module Manager
    • Choose “Modules” tab
    • On Pos Staticblock , click "Configure" button.
    • Choose Add New button to add new static block
    • Enter the value of parameters and click "Save" button when you finish.
  2. How to edit Static Block
    • Login your prestashop admin
    • Navigate to Modules > Module Manager
    • Choose “Modules” tab
    • On Pos Staticblock , click "Configure" button.
    • Choose Static Block you want to edit
    • Change the value of parameters. If you want to change content, please click Source Code button to show code HTML of this block. You can change the content by editing HTML.
    • Click “Save” button to save changes

    • Login your prestashop admin

Megamenu Customer

A. Create new menu item

1. General Setting
  • Login your Prestashop admin
  • Go to Modules > Modules Manager
  • Choose “Modules” tab
  • On Megamenu custommer module, click "Configure" button.
  • In part MENU SETTINGS

  • We got:

    • Background type: Select background type

      - None: Don't use background type

      - Color

      • Background color: Set color for background

      - Image

      • Image source: Upload image background from computer
      • Background repeat: Background image can be repeated along the horizontal and vertical axes, or not repeated at all
    • Sticky menu: Select Yes to enable Sticky menu
    • Center main menu: Select Yes to center align the Main Menu
    • Submenu animation: Select animations for dropown menu items
    • Custom CSS: Add custom CSS for menu item
  • Click Save button when you finish
2. How to build a simple menu item?
  • Login your Prestashop admin
  • Go to Modules > Modules Manager
  • Choose “Modules” tab
  • On Megamenu custommer module, click "Configure" button.
  • Click the Add new item then add the info for the menu item.

  • We got:

    • Title: Add menu title (>>> If use Prestashop link and want to use Prestashop title, leave it empty.)
    • Type Link: Select menu item type (PrestaShop Link / Custom Link / None)

      - PrestaShop Link: Select link from Category, CMS, Supplier, Manufacturer or Page.

      - Custom Link: Enter internal or external link

    • Sub Title: Enter sub title
    • Use icon: Choose icon image

      - No: Don't use icon

      - Font-Awesome Icon: Use font-awesome icon (ex: <i class="fab fa-angellist"></i>)

      - Image Icon: Click Select image > to upload icon from computer

    • Main link style: Config CSS for menu item (ex: color, color hover, background color...)
    • Subtitle style: Config CSS for sub title (ex: color, background color, font size...)
    • Submenu type: There are 3 types of submenus: Mega, Flyout and None (Flyout only is available when you select category in Type Link)
    • Add specific class: Class to style the menu item
    • New window: Choose Yes/No to open new window/tab or not by clicking
    • Active: Choose Yes pulish the menu item
  • Click Save button when you finish
3. How to build columns in Sub-menus?

In here, you can set the width of the sub-megamenu, add column, and add class that style for it. To use this setting, select the menu item as Mega type in Submenu type field.

  • Login your Prestashop admin
  • Go to Modules > Modules Manager
  • Choose “Modules” tab
  • On Megamenu custommer module, click "Configure" button.
  • Click the Build SubMenu then add the info for the menu item.
  • Click on Add a row and then click Save to create new row
  • Click on Add Column to add column for this row

  • We got:

    • Width: Configure column-width. The maximum of columns is 12 based on the bootstrap columns format.
    • Column title: Enter title of column (>>> If use Prestashop link and want to use Prestashop title, leave it empty.)
    • Type Link: Select type for column title(PrestaShop Link / Custom Link / None)

      - PrestaShop Link: Select link from Category, CMS, Supplier, Manufacturer or Page.

      - Custom Link: Enter internal or external link

    • Specific class: Class to style the column title
    • Active in mobile: Choose Yes pulish in the mobile
  • Click Save button when you finish
  • After that, you need configure submenu item

  • We got:

    • Type item: Select submenu item type (Category tree, Prestashop Link, Custom Link, Product, Banner image, HTML block, Manufacturer logo)

      - Category tree

      • Select a category: Select category

      - Prestashop Link

      • Prestashop link: Select link from Category, CMS, Supplier, Manufacturer or Page.

      - Custom Link

      • Title: Enter custom block title
      • URL: Enter internal or external link

      - Product

      • Product: Choose product

      - Banner image

      • + Image source: Upload banner image from computer
      • + Image link: Enter internal or external link for banner image

      - HTML block

      • HTML content: Enter the HTML to define the column

      - Manufacturer logo

      • Select manufacturer: Select manufacturer logo from list
    • Active in mobile: Choose Yes pulish in the mobile

How to set days for new products

  • Login your prestashop admin
  • Navigate to Shop Parameters > Product Settings

How to set a product as discount product
  • Login your prestashop admin
  • Navigate to Catalog > Products
  • Choose item you want to edit, click Edit
  • Choose tab “Prices”
  • From the Prices data tab, look for a checkbox with the description "Display the 'on sale' icon on the product page, and in the text found within the product listing" Click on the checkbox to activate the 'On Sale' status.
  • Click on the Save and Stay button.
  • Click "Save" button

Translate Eposi theme

Modules

  • Login your prestashop admin
  • Go to International > Translations
  • At "Modify translations" section:
    • Type of translation : Translate anything in Prestashop (and not just modules)
    • Select your module : Select the module that you want to translate
    • Select your language : Choose language you want to translate
  • Choose Modify
  • Then Search titles or text need translate
  • After typing your language, click Save and stay or Save to finish.
  • In the footer, you can choose another module for translation.
  • Note: To find more informations, please read on http://doc.prestashop.com/display/PS17/Translations

Staticblock content

  • Login your prestashop admin
  • Navigate to Modules > Module Manager
  • Choose “Modules” tab
  • On Pos Staticblock or Pos Static Footer module, click "Configure" button.
  • Click "Edit" button of your selected block.
  • Choose Source Code to show code HTML of this block in English. Then copy source code.
  • Click to choose language you want to translate
  • Choose Source Code and paste code HTML. Then translate
  • Click Save button when you finish

Example: In Eposi Layout 1, you want to translate text Copyright Posthemes. All Rights Reserved of Copyright from English to Arabic.

  • Login your prestashop admin
  • Navigate to Modules > Module Manager
  • Choose “Modules” tab
  • On Pos Static Footer module, click "Configure" button.
  • At Copyright, click "Edit" button.
  • Choose Source Code to show code HTML of this block in English. Then copy source code.
  • Click to choose Arabic
  • Choose Source Code and paste code HTML. Then translate to Arabic
  • Click Save button when you finish

Translate text in Pos Slideshow Slider

  • Login your prestashop admin. Click "All shop", select the theme you want to translate
  • Navigate to Modules > Module Manager
  • Choose “Modules” tab
  • On Pos Slideshow Slider module, click "Configure" button.
  • At SLIDES LIST section, click "Edit" button of your selected slide.
  • Click of each value of parameters (Slide title, Caption or Description) to choose language then translate.
  • Enter your language & click Save button when you finish
